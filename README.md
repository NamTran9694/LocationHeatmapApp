# Location Heatmap App

A cross-platform mobile application built with C# and .NET MAUI that tracks user location and visualizes movement using a heatmap.

## Features
- Real-time GPS tracking
- SQLite local database storage
- Interactive map visualization
- Heatmap simulation using overlapping circles
- Adjustable radius for visualization

## Technologies Used
- C#
- .NET MAUI
- SQLite
- MAUI Maps
- Android Emulator

## How to Run
1. Install .NET MAUI
2. Clone the repository
3. Run:   
dotnet build -t:Run -f net10.0-android
